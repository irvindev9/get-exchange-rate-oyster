Oyster - Exchange Api

/api/token/ Description: Get a token for the exchange api.

Usage: /api/token?username=oyster&password=oyster123
Add the token to the header of the get request.

Authorization: Bearer #access#

/api/token/refresh/ Description: Refresh the auth token

Usage: /api/token/refresh?token=#refresh#

/api/ Description: Get the exchange rates from 3 sources.

Usage: /api (Add the Authorization header with the token)

Use postman to test the api.
